Methodology

<%= require('./indentation.html') %>

Units

You should use percent (%) to specify width.

You should use rem for text size, element heights, paddings and margins.

File Structure

  • src all the files used to create the application
    • images images used by the application
    • sass style files
    • pages the pages of the application
    • libs supporting libraries written by us
  • node_modules the installed packages
  • package.json programmatic description of the project
  • README.md description of the project

Preprocessor

SASS

Naming convention

BEM (Block__Element--Modifier)

  • btnConfirm
    • btnConfirm__icon
    • btnConfirm__caption
